## What is OpenToast?

OpenToast is a personal pull request attention assistant for teams using GitHub and Slack. It helps you see which pull requests need your attention, then delivers reminders or digests so review work does not disappear into notification noise.

OpenToast is for developers who want a clearer review queue and teams that want pull requests to keep moving without relying on everyone manually checking GitHub all day.

## Current product status

OpenToast is early. The hosted onboarding flow is not documented yet, and parts of the product experience are still being shaped. These docs start with the product concepts and intended user journey while implementation continues.

For local development, backend setup, checks, and test commands, use the [repository README](https://github.com/RossBossDev/opentoast#readme).
